# ch5.me

Static Astro landing page for CH5 LLC.

## Stack

- Astro static site
- React island for the homepage visual
- Three.js hero animation

## Commands

- `pnpm dev` — start local development
- `pnpm build` — build the production site
- `pnpm preview` — preview the built output
- `pnpm check` — run Astro checks

## Purpose

This site gives CH5 a lightweight public presence that matches the company description used for vendors,
business setup, and early product launches.
